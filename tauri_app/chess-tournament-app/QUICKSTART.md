# Quick Start Guide

## Installation

### Windows
1. Download the `.msi` installer from releases
2. Double-click to install
3. Launch "Chess Tournament Manager" from Start Menu

### macOS
1. Download the `.dmg` file from releases
2. Double-click to open
3. Drag "Chess Tournament Manager" to Applications folder
4. Launch from Applications

### Linux
1. Download the `.AppImage` file from releases
2. Make it executable: `chmod +x Chess*.AppImage`
3. Double-click to run (or `./Chess*.AppImage` from terminal)

---

## First Time Setup

### 1. Create a Tournament
- Click **"Start New Tournament"**
- Enter tournament name and number of rounds
- Click **"Create"**

### 2. Add Players

#### From Lichess:
1. Click **"+ Add Player"**
2. In the "Search Lichess" field, type a username or real name
3. Results appear as you type
4. Click **"+ Add"** next to any player

#### From Chess.com:
1. Click **"+ Add Player"**
2. Switch to "Chess.com" tab
3. Enter the exact Chess.com username
4. Click **"Lookup"**
5. Click **"+ Add"**

#### Manually:
1. Click **"+ Add Player"**
2. Enter player name and rating
3. Click **"Add"**

### 3. Generate Rounds
1. Once you have players, click **"Generate Round"**
2. The app will automatically pair players
3. Results appear in the "Rounds" tab

### 4. Enter Results
- Click on a round to see pairings
- For each game, click **W** (White wins), **D** (Draw), or **B** (Black wins)
- Standings update automatically

## Tips & Tricks

1. **Search Tips:**
   - Lichess: Search by username (e.g., "hikaru") or real name (e.g., "Nakamura")
   - Chess.com: Use exact username (case-insensitive)

2. **Rating Selection:**
   - Lichess: Uses Classical rating if available, otherwise Rapid
   - Chess.com: Uses Rapid rating if available, otherwise Blitz

3. **Tiebreaks:**
   - Default tiebreaks: Buchholz and Sonneborn-Berger
   - Customize in tournament settings

4. **Backup Your Data:**
   - Tournament data is saved locally in your browser
   - Regularly export or backup your tournaments

## Troubleshooting

### "Player not found"
- Check spelling of username
- Try searching with a different name
- Ensure you have internet connection

### "API Error"
- Check your internet connection
- Lichess and Chess.com APIs might be temporarily unavailable
- Try again in a few moments

### App crashes on startup
- Try restarting the application
- Check that you have latest version
- On Linux, ensure all dependencies are installed
